# 🏦 CLI Bank Management System

A command-line based Bank Management System built using Python, implementing core banking operations with a modular and object-oriented design.

---

## 🚀 Features

* Create Account with validation (minimum balance, 4-digit PIN)
* Deposit Money with input validation
* Withdraw Money with balance & PIN verification
* Check Balance securely
* View Account Details with transaction history
* Clean CLI interface with structured flow

---

## 🧠 Tech & Concepts Used

* Python (Core)
* Object-Oriented Programming (OOP)
* Classes & Objects (`Account`, `Bank`)
* Exception Handling
* Input Validation
* CLI Design using loops & control flow

---

## 🏗️ Project Structure

```
bank-management-system/
│
├── account.py   # Handles account operations
├── bank.py      # Manages multiple accounts
├── main.py      # CLI interface
```

---

## ▶️ How to Run

python main.py
