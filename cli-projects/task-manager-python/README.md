# Task Manager App
A python CLI-based Task Manager

##Features
>Add Tasks |
>Delete Tasks |
>View Tasks |
>Search Tasks |
>Filter Tasks by priority & status |
>Update Tasks.

## Concept Used
>>List and Dictionaries |
>>Loops and Conditionals |
>>Functions (Both in-built and declared) |
>>Input Validation.
